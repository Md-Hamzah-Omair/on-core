# Product

A privacy-first browser extension that captures user-approved webpages,
extracts useful text, creates embeddings locally, stores them in IndexedDB,
and supports offline semantic search.

# Desktop MVP

- Save the active page through an explicit user action
- Extract title, URL and readable content
- Generate embeddings locally
- Store pages and chunks locally
- Search from a full-page extension dashboard
- Continue working without internet access

# Not in initial scope

- PWA
- Mobile sharing
- GitHub/WebDAV sync
- Accounts
- Cloud inference
- Automatic capture of every page
