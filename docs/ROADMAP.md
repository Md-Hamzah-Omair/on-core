# Roadmap

## Milestone 1 — Runnable Extension Foundation (current)

- WXT, React, strict TypeScript, pnpm, ESLint, and Vitest foundation
- Manifest V3 popup, extension-owned dashboard, and minimal background worker
- No host access or browser permissions
- Disabled content-script placeholder for a later milestone

Capture, extraction, local persistence, AI inference, and semantic search remain
future milestones and are not part of this foundation.
